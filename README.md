# People Management System (Angular + Node.js + MongoDB)

A full-stack CRUD application to manage people records. The application allows users to add, view, update, and delete person details through a clean and responsive interface.

---

## 🚀 Features

- View all people
- Add a new person
- Edit existing person details
- Delete a person
- RESTful API integration
- Real-time UI updates

---

## 🧠 Tech Stack

**Frontend:**

- Angular 8
- TypeScript
- HTML/CSS

**Backend:**

- Node.js
- Express.js

**Database:**

- MongoDB
- Mongoose

---

## 🏗️ Project Structure

- client → Angular frontend
- server → Node.js backend
- models → MongoDB schemas

---

## 🔌 API Endpoints

| Method | Endpoint    | Description         |
| ------ | ----------- | ------------------- |
| GET    | /person     | Get all people      |
| POST   | /person     | Create a new person |
| PUT    | /person/:id | Update a person     |
| DELETE | /person/:id | Delete a person     |

---

## 📦 Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/keerthan-poovaiah/people-management-system.git

---

### 2️⃣ Backend Setup

cd server
npm install
node server.js

---

### 3️⃣ Frontend Setup

cd client
npm install --legacy-peer-deps
ng serve

---

## 🌐 Application URLs

- Frontend: http://localhost:4200
- Backend: http://localhost:3000
